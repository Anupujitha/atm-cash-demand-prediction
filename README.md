# ATM Cash Demand Prediction & Route Optimization

## Overview
This project predicts ATM cash demand using Machine Learning and optimizes refill routes using Google OR-Tools.

## Features
- ATM cash demand prediction
- Refill decision automation
- Route optimization
- Interactive map visualization
- Distance calculation using Haversine formula

## Tech Stack
- Python
- XGBoost
- OR-Tools
- Folium
- Pandas
- Matplotlib

## Workflow
1. Load ATM transaction dataset
2. Train ML model using XGBoost
3. Predict ATM cash demand
4. Identify refill-required ATMs
5. Optimize refill route
6. Generate interactive route map

## Output
- Predicted ATM demand
- Optimized refill path
- Interactive HTML map visualization

## Future Improvements
- Real-time ATM monitoring
